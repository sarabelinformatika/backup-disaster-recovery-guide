# Code of Conduct

## Our Commitment

This project aims to provide a welcoming, respectful and professional environment for everyone.

We encourage constructive discussions, knowledge sharing and technical collaboration.

## Expected Behavior

Please:

- Be respectful.
- Be professional.
- Focus on technical discussions.
- Help improve the documentation.
- Provide constructive feedback.

## Unacceptable Behavior

The following behavior will not be tolerated:

- Harassment
- Personal attacks
- Hate speech
- Spam
- Malicious contributions
- Deliberate misinformation

## Enforcement

Project maintainers reserve the right to remove inappropriate content and reject contributions that do not follow these guidelines.

## Contact

Questions regarding this Code of Conduct can be submitted through GitHub Issues.
